# Recensionsroboten — Veckans möjlighet (Nexar Academy)

AI-svar på Google-recensioner – skräddarsytt för svenska restauranger och deras SEO-ranking 2026.. En **researchad marknadsmöjlighet** redo att bygga.

## Öppna i Claude Code

**Desktop-app** (klistra in i webbläsarens adressfält):
```
claude://code/new?folder=~%2FNexarProjekt%2Frecensionsroboten&q=K%C3%B6r%20exakt%20detta%20i%20mappen%3A%20git%20clone%20https%3A%2F%2Fgithub.com%2Fnexar-mojligheter%2Frecensionsroboten%20.%20(klona%20allts%C3%A5%20in%20i%20den%20H%C3%84R%20tomma%20mappen%2C%20punkten%20p%C3%A5%20slutet).%20L%C3%A4s%20sedan%20CLAUDE.md%20och%20f%C3%B6resl%C3%A5%20en%20konkret%20MVP-plan.
```

**Terminal:**
```bash
git clone https://github.com/nexar-mojligheter/recensionsroboten && cd recensionsroboten && claude "Läs CLAUDE.md och föreslå en konkret MVP-plan."
```

Din Claude Code läser `CLAUDE.md` (instruktioner + projektminne). Hela marknadsresearchen finns i skillen `/affarsplan`.
